# Altos del Torrontés – Hotel Boutique, Wine & Spa

Folleto web (HTML + CSS en línea) listo para publicar en GitHub Pages.

## Estructura
- `index.html` — página principal

## Cómo publicar en GitHub Pages (resumen)
1. Crear repo en GitHub (p.ej. `AltosDelTorrontes`).
2. Subir `index.html` a la raíz del repositorio (rama `main`).
3. Ir a **Settings → Pages** y elegir **Deploy from a branch** → `main` / `/ (root)`.
4. Guardar y esperar a que aparezca el link público.
